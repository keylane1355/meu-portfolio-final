## Experiência em Automação

Desenvolvimento de RPA utilizando Python e Selenium para automação do lançamento de notas fiscais no sistema Fluig.

Principais responsabilidades:
- Captura automática de e-mails com anexos XML e PDF
- Leitura e extração de dados do XML
- Preenchimento automatizado de campos obrigatórios no sistema
- Anexação automática de documentos
- Integração e validação de dados em banco Oracle

Resultados:
Redução do tempo operacional de aproximadamente 3 horas para 40 segundos por nota, com média de 80 notas processadas por dia.

## Objetivo

Atuar como Desenvolvedora Fullstack com foco em automação, integração de sistemas e melhoria contínua de processos.
